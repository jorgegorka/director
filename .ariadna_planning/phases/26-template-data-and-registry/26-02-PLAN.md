Create `app/services/role_template_registry.rb` with the following implementation:

```ruby
class RoleTemplateRegistry
  class TemplateNotFound < StandardError; end
  class InvalidTemplate < StandardError; end

  Template = Data.define(:key, :name, :description, :roles)
  TemplateRole = Data.define(:title, :description, :job_spec, :parent, :skill_keys)

  class << self
    def all
      @templates ||= load_and_validate_templates
    end

    def find(key)
      template = all.find { |t| t.key == key.to_s }
      raise TemplateNotFound, "Template not found: #{key}" unless template
      template
    end

    def keys
      all.map(&:key)
    end

    def reset!
      @templates = nil
    end

    private

    def load_and_validate_templates
      template_files = Dir[Rails.root.join("db/seeds/role_templates/*.yml")].sort
      templates = template_files.map { |file| load_template(file) }
      templates.each { |t| validate_parent_ordering!(t) }
      templates.freeze
    end

    def load_template(file)
      data = YAML.load_file(file)
      roles = data.fetch("roles").map do |role_data|
        TemplateRole.new(
          title: role_data.fetch("title"),
          description: role_data.fetch("description"),
          job_spec: role_data.fetch("job_spec"),
          parent: role_data["parent"],
          skill_keys: role_data.fetch("skill_keys")
        )
      end

      Template.new(
        key: data.fetch("key"),
        name: data.fetch("name"),
        description: data.fetch("description"),
        roles: roles.freeze
      )
    rescue KeyError => e
      raise InvalidTemplate, "Invalid template #{File.basename(file)}: missing #{e.message}"
    end

    def validate_parent_ordering!(template)
      seen_titles = Set.new
      template.roles.each do |role|
        if role.parent.present? && !seen_titles.include?(role.parent)
          raise InvalidTemplate,
            "Template '#{template.key}' has invalid parent ordering: " \
            "'#{role.title}' references parent '#{role.parent}' which has not been defined yet. " \
            "Parents must appear before children in the roles array."
        end
        seen_titles << role.title
      end
    end
  end
end
```

Key design decisions:
- **Data.define** (Ruby 3.2+) for immutable value objects -- lighter than OpenStruct, no need for a full class. Rails 8 ships with Ruby 3.2+.
- **Template and TemplateRole** are frozen value objects -- the registry is read-only data.
- **Memoization via @templates** -- loaded once, cached for the lifetime of the process. `reset!` allows test cleanup.
- **validate_parent_ordering!** iterates roles in array order, tracking seen titles in a Set. If a role references a parent not yet seen, it raises InvalidTemplate. This catches the critical pitfall from v1.5 research.
- **TemplateNotFound and InvalidTemplate** are custom exceptions for clear error handling by consumers.
- **load_template** uses `fetch` for required fields (raises KeyError on missing) and `[]` for optional fields (parent can be nil).
- **sort** on Dir glob ensures deterministic ordering across platforms.
- Class lives in `app/services/` following the project's convention for non-ActiveRecord business logic classes (alongside BudgetEnforcementService, GateCheckService, etc.).
