<objective>
Wire up real-time updates via Turbo Streams and Action Cable so the dashboard updates live when agent status, task status, or audit events change. This implements DASH-04. Uses Rails' built-in Turbo::Broadcastable concern with turbo_stream_from helper for subscription, and after_commit callbacks on models to broadcast changes. No custom JavaScript needed for the Turbo Stream part -- Turbo handles it automatically when the turbo_stream_from tag is present.
</objective>

       - Add private method:
         ```ruby
         def broadcast_dashboard_update
           broadcast_overview_stats
         end

         def broadcast_overview_stats
           company = Company.find(company_id)
           agents = company.agents.active
           Turbo::StreamsChannel.broadcast_replace_to(
             "dashboard_company_#{company_id}",
             target: "dashboard-overview-stats",
             partial: "dashboard/overview_stats",
             locals: {
               total_agents: agents.count,
               agents_online: agents.where(status: [:idle, :running]).count,
               tasks_active: company.tasks.active.count,
               tasks_completed: company.tasks.completed.count
             }
           )
         end
         ```

    5. **Task broadcasts** — Update `app/models/task.rb`:
       - Add `after_commit :broadcast_kanban_update, on: [:create, :update]`
       - Add `after_commit :broadcast_kanban_remove, on: :destroy`
       - Add private methods using remove+append pattern (moves cards between kanban columns):
         ```ruby
         def broadcast_kanban_update
           return unless company_id
           # Remove from current position
           Turbo::StreamsChannel.broadcast_remove_to(
             "dashboard_company_#{company_id}",
             target: "kanban-task-#{id}"
           )
           # Append to correct column
           Turbo::StreamsChannel.broadcast_append_to(
             "dashboard_company_#{company_id}",
             target: "kanban-column-body-#{status}",
             partial: "dashboard/kanban_card",
             locals: { task: self }
           )
         end

         def broadcast_kanban_remove
           Turbo::StreamsChannel.broadcast_remove_to(
             "dashboard_company_#{company_id}",
             target: "kanban-task-#{id}"
           )
         end
         ```

    6. **AuditEvent broadcasts** — Update `app/models/audit_event.rb`:

             "dashboard_company_#{company_id}",

         end
