    1. Create `app/controllers/dashboard_controller.rb`:
       - `before_action :require_company!`
       - `show` action that loads:
         - `@company = Current.company`
         - `@agents = Current.company.agents.active.includes(:assigned_tasks)` (for budget stats)
         - `@total_agents = @agents.count`
         - `@agents_online = @agents.where(status: [:idle, :running]).count`
         - `@tasks_active = Current.company.tasks.active.count`
         - `@tasks_completed = Current.company.tasks.completed.count`
         - `@total_tasks = Current.company.tasks.count`
         - `@total_budget_cents = @agents.where.not(budget_cents: nil).sum(:budget_cents)`
         - `@total_spend_cents = @agents.sum(&:monthly_spend_cents)` (uses agent method, not raw SQL since it depends on period calculation)
         - `@mission = Current.company.goals.roots.ordered.first`
         - `@budget_agents = @agents.where.not(budget_cents: nil).order(:name)` (for per-agent budget cards)

    2. Update `config/routes.rb`:
       - Change `root "home#show"` to `root "dashboard#show"`
       - Add `resource :dashboard, only: [:show]` above root route
       - Keep home controller routes available (don't delete HomeController yet)

    3. Create `app/helpers/dashboard_helper.rb`:
       - `stat_card_trend_class(value)` — returns "stat-card--positive" when value > 0, "stat-card--zero" when 0
       - `budget_summary_percentage(spend, budget)` — returns (spend.to_f / budget * 100).round(1) when budget > 0, else 0.0
       - `tab_link_class(tab_name, current_tab)` — returns "dashboard-tab--active" if matching

       - Stimulus controller with targets: ["tab", "panel"]
       - values: { activeTab: { type: String, default: "overview" } }
       - connect() — call showTab(this.activeTabValue)
       - switch(event) — read data-tab from clicked tab element, call showTab(tabName)
       - showTab(name) — iterate tabTargets: add/remove "dashboard-tab--active" class; iterate panelTargets: show panel where data-tab-name matches, hide others (use hidden attribute)
       - This is a generic tabs controller; use data-action="click->tabs#switch" on tab links

    2. No manual registration needed — the codebase uses `eagerLoadControllersFrom("controllers", application)` in index.js which auto-discovers controllers by naming convention. Just creating `tabs_controller.js` is sufficient.
