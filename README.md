# Director

This website is based on Paperclip.

I was looking for a way of testing Ariadna, a gem that creates Ruby on Rails applications using Claude Code. I saw Paperclip and I thought it is an iteresnting project with a terrible choice of the tech stack so I decided to rewrite it using Ruby on Rails.

The prompt I gave to Ariadna was: "Understand what Paperclip does and how it works. Then, create a plan that implements the same functionality as Paperclip." This is the result.
